Trend

Trend view is where we can get chart display of the data for selected tag(s) for a chosen period of time with an options to display events and event conditions tresholds. Charts look can be customized and reports can be saved.

Add tag(s) to chart

In order for tag(s) data to be displayed tags needs to be added in chart. This is done by selecting an endpoint first from the select list then selecting one or many tags from the multiselect list .

Time period

Tag(s) data can be displayed in the chart for the period selected. We can select the period from the dropdown list “TIME PERIOD” of defined periods or choose a “Custom” period (“Last Day” is the initial preselected period).

When we choose a Custom for a time period, or click to , popup is displayed for start and end date and time to be inserted. This period is applied to the chart after click on “Apply” button.

Edit tag settings

The “Color”, “Thickness” and “Symbol” of the chart tag line can be customized. This can be done via the “Edit tag settings” which are displayed on click on . The change of the “Symbol” (“Round”, “Square” or “TriangleUp”) can be visible only if the “Show chart symbols” is enabled in “Configuration” part.

Delete tags

Tag can be removed from the chart by clicking the delete button in the chart tag .

Reset chart

Chart can be reset by clicking the reset button .

Save Report

Chart with all the tags added can be saved as a Report by clicking the Save button , insert name and choose to save it in the actual period or not.

Load Report

Saved reports can be loaded by clicking on “Open” button . Then we get the “Load report” popup where we choose the report we want to load from the list and then click “Load”.

Edit Report

If you edit an already loaded report then if you like to save it, you would get the “Overwrite” option to choose if you like to overwrite the report or “Save as New” if you like to save a new report.
