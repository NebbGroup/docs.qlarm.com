Trend

×

The Trend View page is where you can get your Tags values-data visualized as Trend. It show a chart display of the data for selected Tag(s) for a specific period of time. You can add Tags and change period while viewing a Report. Also you can get display of Event raised and event thresholds from the Tags’ Conditions. You can customize the chart and have a Reports saved. You can download the Reports as an images and export the chart data in files.

Add tag(s) to chart

In order for Tags data to be displayed Tags needs to be added in chart. This is done by selecting an Endpoint first from the select list “ENDPOINT” then selecting one or more Tags (max 5) from the multiselect “TAGS” list.

Time period

Tags data can be displayed in the chart for the selected period. You can choose the period from the dropdown “TIME PERIOD” list for specified periods or choose a “Custom” period (“Last Day” is the initial preselected period).

When you choose a Custom for a time period, or click , a popup is shown for start and end date and time to be selected. Click “Apply” to apply the time settings in the chart.

Edit tag settings

The “Color”, “Thickness” and “Symbol” of the chart tag line can be customized. Go to “Edit tag settings” with click on the edit button in the Chart Tag.

The change of the "Symbol" ("Round", "Square" or "TriangleUp") can be visible only if the "Show chart symbols" is enabled in "Configuration".

Delete tags

Delete Tags from the chart by clicking the delete button in the chart tag.

Reset chart

Reset chart by clicking the reset button .

Save Report

Chart with all the Tags added can be saved as a Report by clicking the Save button . Insert name and choose to save it in the actual period.

Load Report

Saved Reports can be loaded by clicking the “Open” button . Then you get the “Load Report” popup where you can select the Report you want to load from the list and click “Load”.

Edit Report

If you edit a loaded Report and if you want to save it, you will get the “Overwrite” option in the popup from Fig 11. to choose if you like to overwrite the existing one.

To save the [Report](/glossary#report) as a new [Report](/glossary#report), select "Save as New" option and the loaded [Report](/glossary#report) will remain unchanged.

Chart menu

Chart data can be exported as JSON, CSV, XLSX, HTML, PDF.

Charts can be saved as images, printed and annotated. Images can be saved as PNG, JPG, SVG or PDF.
