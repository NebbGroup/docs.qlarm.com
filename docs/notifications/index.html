Notifications

Notifications view lets us view and filter all of the generated notifications.

The table lists all of the notifications sent to the recipients with their name, type, status, date and recipient information. Using the “Notification Type” drop down list we can filter the notifications by their type (SMS, call, email, push notification). The “Time Filter” drop down list lets us filter the table by the time of the notification. By pressing the button we can further input detail for the time filter.
